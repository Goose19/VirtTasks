Forensics: Encrypted Disk Image -> Key Recovery -> Secret

Дано:

  • disk.img — образ диска с зашифрованным разделом (LUKS2)
  • memdump.raw — дамп памяти системы, работающая с этим диском

Задачи:

  1. Проанализировать memdump.raw
  2. Найти закодированный ключ и декодировать его
  3. Открыть LUKS-раздел
  4. Смонтировать файловую систему
  5. Получить флаг

Формат флага

CTF{...}
