Решение задачи. Местоположение директорий может отличаться. %username% - имя пользователя, указанный вами

  • mkdir solution
  • cd solution
  • cp "/mnt/c/Users/%username%/Downloads/disk.img" /home/%username%/solution
  • cp "/mnt/c/Users/%username%/Downloads/memdump.raw" /home/%username/solution
  • cat memdump.raw (проще открыть в текстовом редакторе, например: notepad++. Ключ находится в самом низу дампа)
  • echo 'ouPG36pW9nzvhya8BZJ7PTdMyy7IOEQwYedHosDAmqc=' | base64 -d > keyfile.bin
  • LOOP=$(sudo losetup -f --show "$(pwd)/disk.img")
  • echo "$LOOP" (выведится /dev/loopN, где N — число)
  • sudo cryptsetup open "$LOOP" luks_ctf --key-file keyfile.bin
  • sudo mkdir -p /mnt/test
  • sudo mount /dev/mapper/luks_ctf /mnt/test
  • cat /mnt/test/secret/flag.txt
